# Acoustic-Echo-Cancellation-in-Hands-Free-Systems
1.Install Python 3.x on your system.
2.Open Notepad / VS Code / Python IDLE.
3.Copy the complete Python code.
4.Save the file.
5.Ensure the file extension is .py.
6.Open Command Prompt / Terminal.
7.Navigate to the folder where Echo.py is saved.
8.Run the program.
9.Press Enter to execute the program.
10.View the output displayed on the screen:
     *Echo power before cancellation
     *Residual power after cancellation
     *Echo cancellation status
